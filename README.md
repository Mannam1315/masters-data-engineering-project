# Masters Data Engineering Project

This repository demonstrates a simple data engineering pipeline combined with API development.

## 🚀 Project Overview

* Built an ETL pipeline using Python
* Processed data from CSV files
* Exposed processed data via Flask APIs

## 📂 Structure

* api/ → API layer
* etl/ → Data processing logic
* data/ → Sample dataset

## ▶️ How to Run

1. Install dependencies
2. Run app.py
3. Open browser: http://127.0.0.1:5000/data

⭐ A graduate research project developed as part of a Master's program, demonstrating end-to-end backend and data engineering capabilities including pipeline design, data modeling, and cloud integration.
